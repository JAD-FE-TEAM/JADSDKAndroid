### Demo 工程结构说明

1、DemoApplication.java 主要用于配置广告SDK初始化配置

2、DemoMainActivity.java 为demo的主页面，用于导航到各种广告页面。

3、com.jd.ad.demo.ex 包下为模板广告相关demo

4、com.jd.ad.demo.na 包下为自渲染相关demo
